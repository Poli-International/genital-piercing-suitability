## What Is Genital Piercing Anatomy Suitability Checker?

The Genital Piercing Anatomy Suitability Checker is an educational reference tool developed by Poli International for tattoo and piercing studios. It provides detailed anatomy suitability information for 8 genital piercing types across male and female anatomy.

The tool presents clinical-grade information including:
- Anatomy suitability percentages and requirements
- Specific suitability tests (e.g., Q-tip test for VCH)
- Standard gauge recommendations (from 16g to 10g)
- Healing time ranges (4 weeks to 12 months)
- Material recommendations (implant-grade titanium, BioFlex®)
- Key risks for each placement
- Contraindications list

The tool uses client-side JavaScript to filter and display information from a predefined dataset. It includes an 18+ age gate and emphasizes that all information is educational only, actual suitability requires in-person examination by an APP-certified piercer.

## Who Should Use This Tool

**Tattoo and Piercing Studio Professionals**
Studio staff can use this tool as a quick reference during client consultations. The standardized information helps ensure consistent messaging about anatomy requirements, healing expectations, and contraindications.

**Clients Researching Genital Piercings**
Adults considering genital piercings can use this tool to understand anatomy requirements before scheduling an in-person consultation. The tool helps set realistic expectations about suitability and healing.

**APP-Certified Piercers**
Experienced piercers can use the tool as a teaching aid during client education sessions. The detailed suitability tests and risk information support informed consent discussions.

## How to Use

### Step 1: Filter by Anatomy Type
Use the "Anatomy type" dropdown to filter piercings by female anatomy, male anatomy, or view all placements. The dropdown defaults to "All placements."

### Step 2: Select a Piercing Type
After filtering, use the "Piercing type" dropdown to select a specific piercing. The dropdown populates based on your anatomy filter selection.

### Step 3: Review Suitability Information
After selecting a piercing type, the tool displays a detailed information card containing:
- Anatomy suitability description with percentage estimates
- Specific suitability test instructions
- Standard gauge and healing time
- Recommended materials
- Key risks
- Contraindications list

## Use Case Examples

### Use Case 1: VCH Consultation
A client asks about VCH piercing suitability. The piercer selects "Female anatomy" from the anatomy filter, then "VCH (Vertical Clitoral Hood)" from the piercing type dropdown. The tool displays:
- Suitability: 70–80% of anatomy suitable
- Suitability test: Q-tip test instructions
- Gauge: 14g (1.6 mm)
- Healing: 4–8 weeks
- Material: Curved barbell, BioFlex® or implant-grade titanium
- Contraindications: Small or low-set clitoral hood, prior scarring, sensitivity disorders

### Use Case 2: PA Piercing Research
A client wants to understand Prince Albert piercing requirements. The piercer selects "Male anatomy" then "PA (Prince Albert)." The tool shows:
- Suitability: Suitable for vast majority of male anatomy
- Gauge: 12g or 10g (2.0–2.4 mm)
- Healing: 4–8 weeks
- Key risk: Urine stream redirection is permanent
- Contraindications: Active UTI, urethral stricture, certain prostate conditions

### Use Case 3: Triangle Piercing Contraindications
A piercer checks triangle piercing suitability for a client with small anatomy. Selecting "Triangle Piercing" reveals:
- Suitability: Only 40–50% of anatomy suitable
- Requires in-person assessment by experienced APP piercer
- Gauge: 12g (2.0 mm)
- Healing: 6–9 months
- Contraindications: Small or recessed clitoral anatomy, insufficient space behind shaft, nerve sensitivity history

## Frequently Asked Questions (FAQ)

### 1. What is the Genital Piercing Anatomy Suitability Checker?
It is an educational reference tool that provides anatomy suitability information for 8 genital piercing types. It covers gauge recommendations, healing times, material suggestions, and contraindications for each placement.

### 2. Is this tool a substitute for an in-person consultation?
No. The tool explicitly states that anatomy suitability cannot be assessed remotely. All information is educational, and actual suitability requires in-person examination by an APP-certified piercer.

### 3. What piercing types does the tool cover?
The tool covers 8 genital piercing types: VCH, HCH, Triangle, Fourchette (female anatomy), and PA, Apadravya, Ampallang, Dydoe (male anatomy).

### 4. What information does the tool provide for each piercing?
Each piercing entry includes: anatomy suitability description, suitability test method, standard gauge, typical healing time, recommended material, key risks, and contraindications.

### 5. How accurate are the suitability percentages?
The percentages (e.g., 70–80% for VCH, 40–50% for Triangle) are educational estimates based on general anatomical studies. Individual suitability varies and requires professional assessment.

### 6. What materials does the tool recommend?
The tool recommends implant-grade titanium for initial piercings and BioFlex® polymer for surface-type and movement-stressed placements. Specific recommendations vary by piercing type.

### 7. Does the tool cover aftercare instructions?
No. The tool focuses on anatomy suitability, gauge, healing time, materials, risks, and contraindications. It does not provide aftercare instructions.

### 8. Why does the tool include an 18+ age gate?
The tool contains clinical information about genital piercings. The age gate ensures users are adults researching or considering genital piercings for educational purposes.

### 9. Can I use this tool to diagnose my own anatomy suitability?
No. The tool is an educational reference only. Self-diagnosis is not recommended. Always consult an APP-certified piercer for in-person assessment.

### 10. How often is the tool updated?
The tool is maintained by Poli International. Updates may occur as industry standards evolve. The current version covers standard APP-recommended practices.
